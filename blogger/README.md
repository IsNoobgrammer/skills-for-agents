# Blogger

Authentic personal-voice writing skill. Produces raw, unpolished, stream-of-consciousness prose that reads like a real human wrote it at 2am — not a corporate blog post.

---

## When to use this skill

- "Write a blog post about X"
- "Express this in my voice"
- "Make this a post"
- "Write like me"
- Any long-form writing where the journey and the honest failure matter as much as the conclusion

---

## Commands

| Command | Effect |
|---------|--------|
| `/blog` | Default mode. 600–1200 words. Balanced voice. |
| `/blog casual` | 300–600 words. More fragments, more slang, voice-note energy. |
| `/blog technical` | 800–1600 words. More code, more numbers, specific benchmarks. |
| `/blog rant` | 400–800 words. Strong opinion, first sentence is the thesis, one fact backs it. |
| `/blog reflection` | 500–1000 words. Human layer, not technical. What the work taught you. |
| `/blog thread` | 8–15 punchy items. X/Twitter thread format. |

---

## What it controls

Blogger owns **voice** — tone, personality, vocabulary, emotional register. It does NOT control structure (that's process skills) or word-count compression (that's density skills).

Drop the `blogger/` folder into your agent's skills directory. Auto-triggers on blog-writing or voice requests.
