## Commands

### Core workflow

| Command | Effect |
|---------|--------|
| `/painter paint` | **Godmode.** Analyze → Diagnose → Plan → Implement → Test → Present. Loops until design scores Good+. |
| `/painter shape` | Plan UX/UI before code. Design brief via discovery interview. |
| `/painter craft` | Full shape-then-build. Blank slate → polished UI with iteration loops. |
| `/painter analyze` | Deep UX review. Heuristics /40, cognitive load, persona walkthrough, slop scan. |
| `/painter polish` | Final pass. 21-item checklist: alignment, spacing, typography, states, motion, a11y. |

### Specialized commands

| Command | Effect |
|---------|--------|
| `/painter audit` | Technical quality. 5 dimensions scored /20. Tags P0–P3. |
| `/painter animate` | Purposeful motion. 100/300/500 rule, expo easing, reduced-motion fallbacks. |
| `/painter colorize` | Strategic color. OKLCH, semantic roles, 60-30-10 weight rule. |
| `/painter layout` | Spatial rhythm. 4pt grid, kill card abuse, gap over margins. |
| `/painter typeset` | Fonts, modular scale, 65–75ch line length, web font loading. |
| `/painter clarify` | UX copy. Error formula, verb+object buttons, empty states. |
| `/painter bolder` | Amplify boring designs. Dramatic scale, weight contrast, asymmetric layouts. |
| `/painter quieter` | Tone down overstimulation. Reduce saturation, flatten nesting, soften motion. |
| `/painter distill` | Strip to essence. ONE primary action, progressive disclosure, 1–2 colors. |
| `/painter harden` | Edge case resilience. Long text, empty, RTL, emoji, i18n, offline. |
| `/painter onboard` | First-run flows. Welcome, empty states, tours (3–7 steps max). |
| `/painter adapt` | Cross-context. Mobile, tablet, desktop, print, email strategies. |
| `/painter optimize` | Performance. Core Web Vitals, images, splitting, critical CSS. |
| `/painter extract` | Pull reusable tokens/components into a design system. |
| `/painter delight` | Moments of joy. Success states, micro-interactions, easter eggs. |
| `/painter overdrive` | Push past conventional. View Transitions, WebGL, scroll-driven animation. |
